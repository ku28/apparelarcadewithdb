How to run this project (Apparel Arcade)

1. Open this folder in Visual Studio Code
2. run "cd Database"
3. run "npm i"
4. run "node .\index.js"
5. run "cd Admin"
6. run "npm i"
7. run "npm start"
8. Add the products or check the product list
9. Go to the main directory
10. run "npm i"
11. run "npm start"
12. Have a look at the website!
